Du an thi ket thuc hoc phan, App Tim Kiem ViTri
Nearby Explore (App Tim Kiem Vi Tri)- Android App

## Giới thiệu
App Tim kiem vi tri(Nearby Explore) là ứng dụng Android giúp người dùng tìm kiếm và chia sẻ địa điểm xung quanh.

## Tính năng
- Đăng ký / đăng nhập
- Hiển thị bản đồ (Google Maps)
- Xem vị trí hiện tại
- Thêm địa điểm
- Hiển thị địa điểm trên bản đồ

##  Công nghệ
- Java (Android Native)
- Firebase Authentication
- Firebase Realtime Database
- Google Maps SDK

## Kiến trúc hệ thống
Android App → Firebase → Google Maps
